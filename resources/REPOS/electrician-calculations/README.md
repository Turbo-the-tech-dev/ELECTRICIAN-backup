# electrician-calculations

All calculations — voltage drop, conduit fill, load calcs, ampacity, bending math.

## Subdirectories

- `voltage-drop/` — Voltage drop formulas and reference
- `conduit-fill/` — Conduit fill calculations and charts
- `box-fill/` — Box fill calculations (NEC 314)
- `load-calcs/` — Load calculations, circuit analysis
- `ampacity/` — Ampacity correction factors, ambient temp adjustments
- `bend-calcs/` — Conduit bending calculations and cheat sheets
- `short-circuit/` — Short circuit current calculations
- `arc-flash/` — Arc flash calculations and apps
- `impedance/` — Impedance, capacitance, inductance, resistance calcs
- `power-calcs/` — Power factor, harmonics, generator/transformer/cable sizing
