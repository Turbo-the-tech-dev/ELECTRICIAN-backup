# electrician-theory

Electrical theory — laws, AC/DC, power, circuits, magnetics, three-phase.

## Subdirectories

- `laws/` — Ohm's, Kirchhoff's, Faraday's, Lenz's, Coulomb's laws
- `ac-dc/` — AC/DC theory, sine waves, RMS, peak values, frequency, wavelength
- `power/` — Apparent, true, reactive power; power triangle, power factor
- `circuits/` — Series, parallel, series-parallel circuits; voltage/current dividers
- `magnetism/` — Electromagnetism, magnetic flux, hand rules, eddy currents, hysteresis
- `reactance/` — Inductive/capacitive reactance, impedance vectors, resonance, phase angle
- `three-phase/` — Delta/wye, high-leg delta, open delta, corner-grounded systems
- `power-quality/` — Harmonics, THD, skin effect, Wheatstone bridge
- `transformers/` — Transformer connections and theory
