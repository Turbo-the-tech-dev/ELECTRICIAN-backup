# ASSETS Directory

Welcome to the ASSETS directory of the ELECTRICIAN project! This directory is designated for storing project assets including images, diagrams, templates, and other media resources.

## Overview

The ASSETS directory serves as the central repository for all visual and media resources used throughout the ELECTRICIAN project. This includes technical diagrams, safety illustrations, educational graphics, and other visual aids that support electrical learning and practice.

## Directory Structure

### Images
- **diagrams/** - Technical electrical diagrams and schematics
- **safety/** - Safety procedure illustrations and PPE images
- **tools/** - Images of electrical tools and equipment
- **installations/** - Photos of electrical installations
- **components/** - Images of electrical components and devices

### Templates
- **forms/** - Standardized forms and checklists
- **reports/** - Report templates for various purposes
- **labels/** - Label templates for equipment and materials
- **signs/** - Safety and identification sign templates

### Documents
- **charts/** - Reference charts and tables
- **specifications/** - Technical specification documents
- **guidelines/** - Implementation guidelines
- **presentations/** - Presentation templates and materials

### Videos
- **training/** - Training video resources
- **demonstrations/** - Installation and procedure demonstrations
- **safety/** - Safety procedure videos
- **equipment/** - Equipment operation videos

## Asset Standards

### Image Specifications
- Preferred formats: PNG, JPEG, SVG, PDF
- Minimum resolution: 300 DPI for print, 72 DPI for web
- Consistent color schemes following project branding
- Clear, legible labeling and annotations

### Naming Convention
- Use descriptive names with hyphens as separators
- Include version numbers where applicable
- Example: `circuit-breaker-types-diagram-v1.png`

### Organization
- Group related assets in appropriate subdirectories
- Maintain consistent folder structures
- Use clear, descriptive names for easy identification

## Usage Guidelines

### For Developers
- Reference assets using relative paths
- Optimize images for web use when applicable
- Maintain aspect ratios when resizing
- Ensure accessibility with alt text

### For Content Creators
- Follow established visual standards
- Ensure technical accuracy in diagrams
- Include appropriate attribution when required
- Maintain consistency with project style

## Contribution

To add assets to this directory:
1. Place files in the appropriate subdirectory
2. Follow the naming convention standards
3. Update this README if new categories are added
4. Ensure files meet quality and format requirements

## Quality Standards

All assets should meet these quality standards:
- High resolution and clarity
- Technical accuracy
- Consistent with project branding
- Appropriate licensing/copyright status
- Optimized file sizes where applicable

## Accessibility

Consider accessibility when creating assets:
- Sufficient color contrast
- Clear, readable fonts
- Alternative text for images
- Compatibility with screen readers

## Maintenance

Regular maintenance includes:
- Removing unused or outdated assets
- Updating assets as standards change
- Verifying asset integrity
- Optimizing file sizes for performance
