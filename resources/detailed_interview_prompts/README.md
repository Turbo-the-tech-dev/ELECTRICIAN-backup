## Detailed Question Types

### Scenario-Based Questions
```
"You are tasked with designing the electrical system for a new manufacturing facility with high-voltage equipment, sensitive electronics, and strict uptime requirements. Walk me through your design process, considering power quality, redundancy, and safety requirements."
```

### Problem-Solving Questions
```
"A critical production line has experienced intermittent electrical failures causing significant downtime. The plant manager is pressuring you to fix it quickly, but initial diagnostics haven't revealed the cause. How would you approach this problem systematically?"
```

### Code Application Questions
```
"Explain how you would apply NEC Article 517 for a new hospital addition that includes patient care areas, operating rooms, and critical care units. What specific requirements would you need to address for each area?"
```

### Safety-Critical Questions
```
"You discover that a colleague has bypassed a safety interlock on electrical equipment to keep production running. The equipment appears to be functioning normally. What is your immediate response and follow-up action?"
```
