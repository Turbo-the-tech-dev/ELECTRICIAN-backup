# CALCULATORS Directory

Welcome to the CALCULATORS directory of the ELECTRICIAN project! This directory contains electrical calculation tools, utilities, and resources to assist with various electrical computations and design tasks.

## Overview

The CALCULATORS directory provides essential computational tools for electrical professionals. These tools help with common electrical calculations including load calculations, voltage drop, conduit fill, and other critical electrical design parameters. Having accurate calculation tools is essential for safe and code-compliant electrical installations.

## Directory Structure

### Load Calculations
- **residential-load-calculator/** - Tools for residential load calculations per NEC Article 220
- **commercial-load-calculator/** - Commercial and industrial load calculation tools
- **motor-load-calculator/** - Motor load calculation tools
- **appliance-load-calculator/** - Appliance and equipment load calculations

### Circuit Protection
- **breaker-sizing-calculator/** - Circuit breaker sizing tools
- **wire-gauge-calculator/** - Wire gauge selection tools
- **overcurrent-protection/** - Overcurrent protection calculation tools
- **transformer-sizing/** - Transformer sizing and selection tools

### Voltage Drop
- **voltage-drop-calculator/** - Voltage drop calculation tools
- **cable-length-calculator/** - Cable length and voltage drop tools
- **conductor-sizing/** - Conductor sizing based on voltage drop

### Conduit Fill
- **conduit-fill-calculator/** - Conduit fill calculation tools

### Power Calculations

### Safety Calculations
- **arc-flash-calculator/** - Arc flash incident energy calculations
