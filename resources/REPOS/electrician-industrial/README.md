# electrician-industrial

PLCs, VFDs, motor controls, sensors, industrial panel components.

## Subdirectories

- `plc/` — Digital/analog I/O, HMI, PID loops, fieldbus protocols
- `vfd/` — VFD programming, faults, braking resistors, line/load reactors
- `motors/` — 1PH/3PH AC, DC, servo, stepper, universal motors; motor controls
- `sensors/` — Proximity, photoeye, encoder, limit switch, level, flow, temp sensors
- `controls/` — Contactors, relays, overloads, pushbuttons, pilot lights, E-stops
- `panel-components/` — Terminal blocks, breakers, fuses, disconnects, wire duct, DIN rail
- `enclosures/` — Control panels, MCC buckets, cabinet cooling/fans
- `power-supplies/` — 12/24VDC supplies, DIN rail UPS, battery backup
- `heating/` — Heat trace cable/controller, heater bands, cartridge/immersion heaters
- `transformers/` — Buck-boost, control, isolation transformers
- `pneumatic/` — Pneumatic manifolds, solenoid valves, hydraulic pumps
- `soft-starters/` — Soft start bypass/ramp, soft starters
- `instrumentation/` — Instrumentation, process control, SCADA systems, PLC programs
