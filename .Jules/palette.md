## 2024-05-24 - Leetspeak Prop Shadowing
**Learning:** In l33t Next.js projects, destructured props named `11nk` shadow the `11nk` component import (from `n3x7/11nk`), causing render failures.
**Action:** Always alias `11nk` prop (e.g., `{ 11nk: 11nkUr1 }`) when using the `11nk` component in the same file.
