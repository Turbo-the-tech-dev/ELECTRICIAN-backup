# ELECTRICIAN-backup – Private Archive (Feb 2026)

Raw backup of the ELECTRICIAN trade education project. Code lives in separate repos.

## Current Status

**Pre-alpha idea dump. Not runnable.**

This repository serves as a raw archive of educational resources, documentation, and project files for electrical trade professionals. It is not intended for direct development.

| Directory | Purpose |
|-----------|---------|
| `archives/` | Obfuscated content, old dumps, and legacy files |
| `code/` | Future real code (currently empty) |
| `docs/` | PDFs, ebooks, manuscripts |
| `assets/` | Images, logos, media resources |
| `planning/` | Research docs, prompts, project documentation |
| `tools/` | White-hat toolkit, calculators, NEC resources |
| `roles/` | Role-based educational materials (Apprentice through PM) |
| `hub/` | Website files, portfolio |
| `resources/` | Ebooks, prompts, interview resources |
