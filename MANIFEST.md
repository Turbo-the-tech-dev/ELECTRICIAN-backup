# MANIFEST

Directory manifest for ELECTRICIAN-backup repository.

| Directory | Purpose |
|-----------|---------|
| `archives/` | Obfuscated content, old dumps, legacy files, and historical logs |
| `code/` | Placeholder for future real code (currently contains Electrician app sources) |
| `docs/` | Book manuscripts, ebooks, and long-form documentation |
| `assets/` | Images, logos, icons, and media resources |
| `planning/` | Research documents, AI prompts, project documentation (CLAUDE.md, GEMINI.md, QWEN.md) |
| `tools/` | White-hat security toolkit, electrical calculators, NEC resources, theory apps |
| `roles/` | Role-based educational materials organized by career level (Apprentice, Journeyman, Foreman, Contractor, Estimator, PM, MEP) |
| `hub/` | Website files, portfolio site, index.html |
| `resources/` | Ebooks, prompts, interview resources, REPOS references |

| File | Purpose |
|------|---------|
| `README.md` | Project overview and status |
| `LICENSE` | All Rights Reserved license |
| `SETUP.md` | Setup notice (backup only, not for development) |
| `MANIFEST.md` | This file - directory purpose listing |

| `.github/` | GitHub configuration (workflows, issue templates) |
