# PROMPTS Directory

Welcome to the PROMPTS directory of the ELECTRICIAN project! This directory contains various prompts and queries designed for AI tools like GEMINI, CLAUDE, and other AI assistants to generate electrical-related content.

## Overview

The PROMPTS directory serves as a repository of well-crafted prompts specifically tailored for the electrical industry. These prompts can be used with AI tools to generate educational content, solve technical problems, create documentation, and assist with various electrical tasks.

## Prompt Structure

### Effective Prompt Components
1. **Role Definition** - Specify the intended audience (apprentice, journeyman, contractor)
2. **Context** - Provide relevant background information
3. **Specific Request** - Clearly state what information is needed
4. **Format Requirements** - Specify desired output format
5. **Constraints** - Include any limitations or requirements

### Example Prompt Template
```
Act as a master electrician. Explain [specific electrical concept] to an apprentice electrician. Include:
- Basic definition
- Practical applications
- Safety considerations
- Relevant NEC code references
Provide the information in a clear, step-by-step format with examples.
```
