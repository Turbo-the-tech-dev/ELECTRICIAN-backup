# electrician-nec

NEC code reference — articles, annexes, chapter 9 tables, NFPA standards, historical versions.

## Subdirectories

- `articles/` — NEC articles (Art. 100–840)
- `annexes/` — NEC annexes A–J
- `chapter9/` — Chapter 9 tables 1–12
- `examples/` — Code examples D1–D13
- `standards/` — NFPA 70E, 72, 79, 101, 110; IEEE color books
- `amendments/` — City/state amendments, TIAs, errata, formal interpretations
- `historical/` — NEC editions: 2011, 2014, 2017, 2020, 2023
- `index/` — Alphabetical index sections A–Z
- `reference/` — Code references, compliance notes, definitions
